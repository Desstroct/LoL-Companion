How it works

Shows the current TFT meta team compositions, sourced from tftactics.gg.

Key press: Cycle to the next comp.
Dial rotate: Scroll through comps.
Dial press: Cycle tier filter (All → S → A → B).
Touch: Force refresh data (re-fetches from web).

Each comp shows: tier, comp name, playstyle (Fast 8, Slow Roll, etc.), champions, and key carry items.

Data is cached for 2 hours to avoid excessive requests. Works in both TFT and LoL modes — you can browse comps anytime.
