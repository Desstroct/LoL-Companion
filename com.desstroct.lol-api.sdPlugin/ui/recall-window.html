How it works

Shows a recall timer based on your current gold and item breakpoints.

RECALL NOW — You have enough gold for a meaningful purchase (component or full item).
SAVE — You're close to a breakpoint, keep farming.

The gold target setting lets you customize the minimum gold needed before suggesting a recall (default: 1100g — typical first component cost).

Gold breakpoints are dynamically calculated from your recommended build when available.

Data from the Riot Live Client API (in-game only).
