How it works

Automatically switches Stream Deck profiles based on the current League of Legends game phase.

Phase → Profile mapping:
• Lobby / Queue / Ready Check → LoL Lobby
• Champion Select → LoL Champ Select
• In Game / Loading → LoL In Game
• Client closed → stays on current profile

Press the key to toggle auto-switching ON/OFF.

The three LoL profiles are installed with the plugin. You can freely rearrange the keys on each profile — they are NOT read-only.
